# Install on linux
For app installed sqlite and enabled cookies required<br>
``` 
mkdir quiz
```
Move all files to quiz folder
```
rm -rf venv
python -m venv "env"
source env/bin/activate
pip3 install -r requirements.txt
```
# Run
```
python -m uvicorn main:app
```
